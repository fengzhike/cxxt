1.关于创新学堂

1.1使用创新学堂能做什么？
      创新学堂客户端是面向学生创新创业教育的互联网实训平台应用，在这里学生可以在共享开源课程、网上互动交流学习的同时，免费发布、查询、浏览服务与需求，进行真实的交易，应聘真实工作岗位，获得真实企业与投资人的关注。

1.2使用创新学堂是否收费？
      创新学堂对学生用户不收取管理服务费，对企业与机构用户收取一定的管理与信息服务费，具体条款以用户协议为准。

1.3如何注册创新学堂帐号？
      首页点击“我的”，进入“我的”页面，点击上方登录，进入登录/注册界面，点击进入注册页面，按照要求如实填写信息即可完成创新学堂帐号的注册。

1.4 个人能力（PR）值是什么？
      PR值是指当前用户在某服务标签下的个人能力值。能力值越高，服务可获得更大的购买几率及招聘企业的关注度越高。

1.5 什么是经营绩效指标？
      绩效指标包括当月收入，累计收入，资产收益率（月），服务分数，发布数量，购买服务数量等，学员可以通过实践理解绩效指标的各项内容，学会管理自己的财富，提升创业能力。

1.6 什么是五色思维评价？
      五色思维是生命思维，批判思维，美学思维，经济思维，设计思维五类创创新思维。通过学习实践互动，创新学堂通过大数据分析出人的五色思维相对分数作为人才评价参考。“近朱者赤“，你与其他师生在创新学堂互动越多，这些师生思维能力越强，你的思维评价也会越高。


2.创新学堂操作相关

2.1 如何发布信息？
      首先登陆，在首页点击“发布”（APP为首页下方中间按钮，网页为右上角“发布”图标），可以发布服务，需求，课程，活动等信息。发布后经过审核的信息可以被其他用户查找到。

2.2 服务发行后还能修改信息吗？
      服务发行后，可在“我的“—”发布的服务/需求/…“中针对已发服务进行编辑，若此服务已被购买，则不会改变订单内容，按未修改前服务信息进行支付交易。

2.3 信息发布后是否还能退市
      可以，服务卖出后，可在“我的”—“发布的服务/需求/…”中进行信息的上市/退市操作。

2.4 如何进行信息搜索
      首页中点击搜索功能进入搜索页面，您可直接点击产品默认分类进行相关分类标签信息搜索，也可在搜索输入框中输入搜索关键字进行搜索，在搜索结果页面您可以切换分类栏选择您想要搜索的关键字数据分类。

2.5 如何在有名市场进行交易？
      首先查找到服务，决定交易后点击“立即购买”，提交订单即可与服务提供者进行交易。购买成功的服务可以在首页“订单”中查询到。

2.6 服务被购买后还能修改价格吗？
      服务发布之后，若与买家沟通后需要针对价格进行修改，在买家提交订单未进行支付（或调取第三方支付页面），则可在“我的”，“发布的服务”—“买家待付中“针对价格进行修改，买家可在被提示价格更改后支付修改后的价格购买服务。

2.7 如何听课与查阅课程资料？
      在平台首页未名讲堂板块，点击课程即可查阅创新创业课程视频，进行学习，同时也可以和身边的同学或教师进行平台内交流互动，平台支持通信功能。

2.8 如何收藏别人的服务或需求？
      在服务或需求详情里，右上角的位置有收藏按钮，点击收藏即可将该服务或需求收藏至你的个人收藏里，方便你随时查阅。

2.9 用户是否可以删除已经发布的信息
      不可以，创新学堂不提供服务删除功能，用户可以在“我的—发布的服务/需求/…”里面将已经发布的信息进行退市操作。

2.10 为什么我的服务被退市了？
      如果您发布的服务有不合法或者是不合规的情况，创新学堂管理员有权利将您所发布的违规信息进行锁定处理，（情节严重者将进行帐号锁定、及保留向公安机关举报处理的权利。）进行相应修改后可重新上市。

2.11 发票的处理
      创新学堂上的服务交易所对应的发票的处理由买卖双方自行商量确定，创新学堂目前不负责具体发票的事宜。

2.12 我是购买者，如果对方服务不满意怎么办？
      交易采用完成后确认付费制。服务实际完成后，购买者需要在“订单”页面，”待确认“订单中点击确认，对方才会收到付款。如果”不确认“的订单在一定时期后会自然失效。

2.13 我是服务者，如果对方购买后一直不确认怎么办？
      服务者最好在服务进行前与对方确定具体的服务交付办法与确认时间，在服务进行后立刻提醒请购买者确认支付。

2.14 如果遇到交易争议或者不实交易怎么办？
      在交易之前一定要确定对方是否是实名认证过的用户。对不满意的服务可以在评价时如实反映。对不实信息或违反创新学堂平台用户规定的行为可以进行投诉。对认定违法行为可以直接请律师联系对方或向公安机关进行报案。


3.关于账户

3.1 如何获得认证
      在首页点击“我的”，在“我的”页面点击自己的姓名进入“账号管理”，在“身份认证”中上传自己个人身份证明可以获得身份认证，在“银行卡认证”中进行测试小额转账可以获得银行卡认证。创新学堂对认证信息将严格保密，第三方与其他用户无法看到。

3.2 如何添加更新个人信息？
      在首页点击“我的”，在“我的”页面点击自己的姓名进入“账号管理”，进入“修改基本信息”添加更新自己的个人信息。在最下方有“所在学校”一项可以添加学校名称。


4.关于财务

4.1 如何为我的账号进行充值？
      在首页点击“我的”，在“我的”页面点击自己的姓名进入“账号管理”，进入“充值”可以输入充值卡与密码进行充值。购买充值卡请联系课程助教创新学堂客服。

4.2 如何进行提现
      用户登录后，在提现页面，填写银行卡号、开户银行、开户姓名、提现金额、提交相关信息，公司运营服务部确认后在3个工作日内将资金汇款到提现用户指定的银行帐号（载转账过程中涉及的银行服务费予以扣除）
